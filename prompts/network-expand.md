# Prompt: `network-expand`

When Nick types `network-expand`, load `skills/network-expansion/SKILL.md` and execute.

Steps:
1. Run 3 targeted news searches (stablecoin, maritime, VC activity)
2. Extract and qualify names from results
3. Present the connection queue for Nick's review
4. For each approved name, walk through Cowork-assisted LinkedIn add
5. Update CRM with new rows at Stage 2 + Next Action Date = today+7

Requires Claude Cowork desktop app to be running.
